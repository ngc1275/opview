opview
======

A PHP extension to view opcode of php scripts.Like vld but not only in cli mode!

version
======
v1.0.0

Support
======
    PHP 5.4 / PHP 5.5

install
======
    phpize
    ./configure --with-php-config=/path-of-php/php-config
    make && make install

functions available
======
opview_string(string $php_script);
